### **Étape 3: Tester dans le Navigateur**

2. Cliquez: **"Add New Atelier"**
3. Entrez:
   - Atelier Name: `Atelier Ahmed Ali`
   - Email: `ahmed@example.com`

6. Puis: Redirection automatique vers la liste

## ✅ Si ça Ne Marche Pas

### Ouvrir Console (F12)
```
Ctrl+Shift+K (ou Cmd+Shift+K sur Mac)
```

### Chercher l'erreur:
```
Si vous voyez:
"Backend server is not responding..."
→ Vous avez oublié de démarrer le backend (Étape 1)

Si vous voyez:
"Format must be Atelier FirstName LastName"
→ Vous n'avez pas entré le bon format

Si vous voyez:
"nbrParticipant must be greater than 15"
→ Entrez au moins 16 participants
```

```
Nom: Ahmed Ali  ← MANQUE "Atelier "
Email: not-email  ← Format email incorrect
Participants: 10  ← Moins de 15
```

## 🆘 Problèmes Courants

| Problème | Solution |
|----------|----------|
| "Backend not responding" | Démarrer backend: `npm start` (Étape 1) |
| "Format must be Atelier..." | Entrer: `Atelier Ahmed Ali` pas `Ahmed Ali` |
| "Loading forever..." | Vérifier F12 Console pour l'erreur exacte |
| Database error | Vérifier que le backend a `database.sql` |
| Port 3000 already in use | Changer le port ou tuer le processus |
